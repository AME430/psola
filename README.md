Implements the TD-PSOLA algorithm.


Requirements
------------
- Numpy
- Matplotlib
- Scipy


Basic Usage
-----------

Install from PyPI

    pip install psola

or from the source directory

    python setup.py install

and use the command line script to interface with the package

    psola_exec ...
